## 1. ゲームの厚み（中身が薄い）

coin の **earn** はある。**spend** と「まちの生活」が弱い。

- [ ] **商業施設**（`market_stall` 等 + 配置コスト）
- [ ] **coin の使い道**（種代・肥料・店舗）
- [ ] **施設効果**（近くの畑バフ等 — 商業を置く理由）
- [ ] **NPC**（歩行者 2–3 体からで可）
- [ ] **店舗開閉 × 時間**（`worldTime` 連動）
- [ ] **電車**（`rail` は見た目のみ → 乗れる transport）
- [ ] **バス路線**（Line 不満・path と連動）
