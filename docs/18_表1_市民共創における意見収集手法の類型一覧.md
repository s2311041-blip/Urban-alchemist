## 表1本体（手法類型表）

| 類型 | 含まれる手法・ツール・場（具体例） | 目的適合比較表で比べる代表 |
|------|-----------------------------------|---------------------------|
| **1. 集団対話型** | ワークショップ（対面）、オンライン／ハイブリッドWS、タウンミーティング、世界カフェ、チャレンジワークショップ、プレゼン・ピッチ（アイデア募集）、住民説明会の対話部分※ | **ワークショップ** |
| **2. 市民参加プラットフォーム型** | みんスタオンライン（産総研等基盤・自治体利用）、柏の葉キャンパス周辺の自治体アプリ、自治体まちづくりアプリ／デジタル意見箱、FixMyStreet／311系通報 | **みんスタオンライン**（代表例） |
| **3. 空間・地図型** | 参加型GIS（PGIS）、オープンマップ／マッピング、ストリートビュー・写真マップへの投稿、ウォーキング調査＋記録※ | **参加型GIS** |
| **4. 個人・非同期・モバイル型** | SNS（X／Instagram等）、LINE公式・グループ、非同期掲示板、写真＋位置情報アプリ、ハガキ・投函箱（物理） | **SNS** |
| **5. 記述調査型** | Webアンケート／Googleフォーム、郵送・対面住民アンケート | **Webアンケート** |
| **6. 活動埋め込み型** | Minecraft等によるまちづくりワールド、ゲーム内投稿（他プロジェクト）、本研究 Urban Alchemist の投稿機能※ | （目的適合比較表に載せず・RQ2） |
| **7. 没入・仮想空間型** | メタバース上の意見交換、VR空間での意見共有イベント、VRによる現地再現体験（副次） | （手法類型表のみ・RQ2／展望） |
| **8. 本研究の提案** | **非同期AR投稿（空間注釈）** → RQ1／**参加型ゲーム（視点置換）** → RQ2 | **非同期AR**（比較結論の主張対象） |

※ 境界例：単体では目的適合比較表に載せないが、類型の理解のため表1に記載。

## 本研究のスコープ外（除外一覧・脚注用）

除外基準 E1〜E4 を適用したもの。手法類型表には **載せない**。

| 手法 | 除外基準 | 理由（短く） |
|------|----------|--------------|
| パブリックコメント（法定） | E1 | 一期限りの計画手続 |
| 住民投票・選挙 | E2 | 意見の自由記述・収集が主でない |
| マーケティング調査 | E3 | まちづくり・市民共創の文脈外 |
| 研究者による構造化インタビュー | E4 | 比較前提が異なる |
| 長期民族誌・フィールドノート | E4 | 同上 |
| フィールドワーク＋振り返り（研究者主導） | E4 | 同上 |
| 行政内部ヒアリング | E2, E3, E4 | 市民向け収集ではない |
| 広報・プレスリリース（一方向） | E2, E4 | 収集ではなく発信 |
| 住民説明会（聴講中心） | E1, E2 | 意見収集より情報提供 |
| ポストイット（WS内の技法のみ） | E4 | 単体手法として扱わない |
| 本研究ゲーム（RQ2の提案） | — | 比較対象ではなく提案側 |
